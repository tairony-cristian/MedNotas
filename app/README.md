# MedNotas

**MedNotas** é uma aplicação desenvolvida em Python usando PyQt5 e SQLite para o gerenciamento de anotações médicas. O sistema permite adicionar, editar, excluir e pesquisar anotações, além de gerar relatórios.

## Funcionalidades

- **Gerenciamento de Anotações:** Adicione, edite e exclua anotações médicas.
- **Pesquisa:** Busque anotações por ID, período de data, local, médico e nota fiscal.
- **Exportação de Relatórios:** Gere relatórios com os dados das anotações.
- **Interface Gráfica:** Interface intuitiva desenvolvida com PyQt5.

## Tecnologias Utilizadas

- **Python** (versão 3.x)
- **PyQt5** (para a interface gráfica)
- **SQLite** (para o banco de dados)

## Uso
Interface Principal
Adicionar Anotação: Clique no botão "Adicionar" para abrir o diálogo de entrada de dados.
Editar Anotação: Selecione uma anotação na tabela e clique em "Editar" para modificar os dados.
Excluir Anotação: Selecione uma anotação na tabela e clique em "Excluir" para removê-la.
Pesquisar: Use o campo de pesquisa para encontrar anotações com base em diferentes critérios.
Exportar Relatório: Clique no botão "Exportar Relatório" para gerar um relatório com os dados da tabela.
